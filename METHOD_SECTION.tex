\section{方法}\label{sec:method}

基于第~\ref{sec:prelim}~节的 PGA 框架，
本节给出与当前实现一致的 Clifford 代数多模态等变网络（CAME-Net）。
网络包含四个核心模块：
多模态射影嵌入（MPE，\ref{sec:mpe}）、
几何 Clifford 注意力（GCA，\ref{sec:gca}）、
几何层归一化（GLN，\ref{sec:gln}）以及软等变正则化（\ref{sec:equiv_reg}）。
在实现上，点云、图像和文本首先分别映射为 $\mathrm{Cl}(3,0,1)$ 中的多重向量 token；
随后所有 token 沿序列维拼接，并送入共享的 GCA 堆栈；
最后通过全局均值池化与标准任务头输出分类结果。

% ------------------------------------------------------------
\subsection{多模态射影嵌入}\label{sec:mpe}
% ------------------------------------------------------------

\paragraph{点云嵌入。}
设输入点云为 $\{\mathbf{p}_i\}_{i=1}^N$，其中 $\mathbf{p}_i=(x_i,y_i,z_i)^\top$。
对每个点，我们先用 $K$ 近邻估计局部几何：
\begin{equation}\label{eq:mpe_knn}
  \mathcal{N}_i = \operatorname{KNN}_K(\mathbf{p}_i), \qquad
  C_i = \frac{1}{K}\sum_{j\in\mathcal{N}_i}
  (\mathbf{p}_j-\mathbf{p}_i)(\mathbf{p}_j-\mathbf{p}_i)^\top.
\end{equation}
记 $C_i$ 的特征分解为
\begin{equation}\label{eq:mpe_eig}
  C_i = U_i \operatorname{diag}(\lambda_{i1},\lambda_{i2},\lambda_{i3}) U_i^\top,
  \qquad \lambda_{i1}\le \lambda_{i2}\le \lambda_{i3},
\end{equation}
则最小特征值对应的特征向量给出局部法向 $\mathbf{n}_i$。
为消除符号歧义，实现中采用全局质心 $\bar{\mathbf{p}}$ 做定向：
\begin{equation}\label{eq:mpe_normal_orient}
  \mathbf{n}_i
  \leftarrow
  \operatorname{sign}\!\bigl((\mathbf{p}_i-\bar{\mathbf{p}})^\top \mathbf{n}_i\bigr)\,\mathbf{n}_i .
\end{equation}

在此基础上构造一个刚体运动不变量描述子
\begin{equation}\label{eq:mpe_invariant_descriptor}
  \mathbf{u}_i =
  \Bigl[
    \lambda_{i1},\lambda_{i2},\lambda_{i3},
    r_i,\;
    \|\mathbf{p}_i-\bar{\mathbf{p}}\|_2,\;
    \frac{\lambda_{i3}}{\lambda_{i1}+\lambda_{i2}+\lambda_{i3}+\epsilon}
  \Bigr],
\end{equation}
其中 $r_i=\frac{1}{K}\sum_{j\in\mathcal{N}_i}\|\mathbf{p}_j-\mathbf{p}_i\|_2$。
实现中，逐点标量通道仅由该不变量描述子以及可选的逐点属性 $\mathbf{h}_i$ 产生：
\begin{equation}\label{eq:mpe_scalar}
  s_i = g_0\!\bigl([\psi(\mathbf{u}_i)\;\|\;\phi(\mathbf{h}_i)]\bigr)
  \in \langle\mathrm{Cl}\rangle_0,
\end{equation}
其中 $\psi,\phi,g_0$ 均为小型 MLP；若无额外点特征，则仅使用 $\psi(\mathbf{u}_i)$。

点云的几何通道采用\emph{闭式} PGA 构造，而非对坐标做自由线性投影。
首先，1 阶分量定义为过点 $\mathbf{p}_i$ 的局部切平面：
\begin{equation}\label{eq:mpe_plane}
  \Pi_i
  =
  n_{ix} e_1 + n_{iy} e_2 - n_{iz} e_3
  - (\mathbf{n}_i^\top \mathbf{p}_i)\,e_0 .
\end{equation}
这里采用了与当前基底顺序一致的对偶平面表示；
其对应的 incidence 关系为
$a x + b y - c z + d = 0$，
因此实现中的 1 阶系数为 $(a,b,c,d)=(n_{ix},n_{iy},-n_{iz},-\mathbf{n}_i^\top\mathbf{p}_i)$。
该构造在 motor 三明治积下严格按平面对象协变变换。

其次，2 阶分量编码局部法向方向本身，
但实现中不使用欧几里得二向量 $\{e_{23},e_{31},e_{12}\}$，
而使用其\emph{理想二向量}形式：
\begin{equation}\label{eq:mpe_ideal_bivector}
  D_i
  =
  \bigl\langle P(\mathbf{p}_i)\,P(\mathbf{p}_i+\mathbf{n}_i)\bigr\rangle_2
  =
  -n_{ix} e_{01} - n_{iy} e_{02} + n_{iz} e_{03},
\end{equation}
其中 $P(\mathbf{p})$ 表示点 $\mathbf{p}$ 的 OPNS PGA 表示。
由于该分量本质上来自两点几何积的 2 阶部分，
它只依赖位移方向，与全局平移无关，
并在旋转下按方向对象精确协变。

最后，3 阶分量直接使用点的精确 OPNS 表示：
\begin{equation}\label{eq:mpe_point}
  P(\mathbf{p}_i)
  =
  e_{123}
  + x_i e_{032}
  + y_i e_{013}
  + z_i e_{021}.
\end{equation}
因此，点云 token 的完整嵌入为
\begin{equation}\label{eq:mpe_pc}
  \mathcal{X}_i^{\mathrm{pc}}
  =
  s_i + \Pi_i + D_i + P(\mathbf{p}_i).
\end{equation}
该实现中不使用 4 阶伪标量点云通道。

\paragraph{图像嵌入。}
对图像 patch 特征 $\mathbf{v}_j$，
实现仅保留标量与伪标量通道：
\begin{equation}\label{eq:mpe_img}
  \mathcal{X}_j^{\mathrm{img}}
  =
  g_0^{\mathrm{img}}(\mathbf{v}_j)
  +
  g_4^{\mathrm{img}}(\mathbf{v}_j)\,I.
\end{equation}
由于标量与伪标量在 motor 三明治积下保持不变，
该嵌入天然为 $\mathrm{SE}(3)$ 不变量。

\paragraph{文本嵌入。}
文本 token 特征 $\mathbf{w}_k$ 的实现与图像完全对称：
\begin{equation}\label{eq:mpe_txt}
  \mathcal{X}_k^{\mathrm{txt}}
  =
  g_0^{\mathrm{txt}}(\mathbf{w}_k)
  +
  g_4^{\mathrm{txt}}(\mathbf{w}_k)\,I.
\end{equation}

\paragraph{多模态拼接。}
实现中，MPE 不在嵌入阶段做交叉注意力或平均融合，
而是将各模态 token 直接沿序列维拼接：
\begin{equation}\label{eq:mpe_concat}
  \mathcal{X}
  =
  \bigl[
    \mathcal{X}^{\mathrm{pc}}
    \;\|\;
    \mathcal{X}^{\mathrm{img}}
    \;\|\;
    \mathcal{X}^{\mathrm{txt}}
  \bigr].
\end{equation}
后续所有跨模态交互统一由 GCA 完成。

\subsection{几何 Clifford 注意力}\label{sec:gca}
% ------------------------------------------------------------

对任意输入多重向量序列 $\{\mathcal{X}_i\}$，
每个注意力头都对每一阶分量独立施加线性映射，
从而得到 grade-preserving 的查询、键和值：
\begin{equation}\label{eq:qkv}
  \mathcal{Q}_i = W_Q \mathcal{X}_i, \qquad
  \mathcal{K}_j = W_K \mathcal{X}_j, \qquad
  \mathcal{V}_j = W_V \mathcal{X}_j.
\end{equation}
注意力分数由几何积的标量部分给出：
\begin{equation}\label{eq:attn_score}
  a_{ij}
  =
  \frac{1}{\sqrt{d_{\mathcal{M}}}}
  \bigl\langle \mathcal{Q}_i \widetilde{\mathcal{K}}_j \bigr\rangle_0 .
\end{equation}
根据 PGA 几何积的协变性，
当所有输入同时经历同一个 motor 变换时，
$a_{ij}$ 保持不变。

加权输出写为
\begin{equation}\label{eq:attn_output}
  \mathcal{O}_i
  =
  \sum_j
  \frac{\exp(a_{ij})}{\sum_{j'}\exp(a_{ij'})}\,
  \mathcal{V}_j,
\end{equation}
其中 softmax 权重为标量，因此输出与 $\mathcal{V}_j$ 共享相同的协变类型。
实现采用多头机制；每个头拥有独立的 per-grade 线性映射，
各头输出在通道维拼接后，再由一组 per-grade 输出映射回 16 维 PGA 系数。
当输入为多模态 token 拼接序列时，
GCA 同时完成同模态建模与跨模态交互。

每个 CAME 层的残差结构为
\begin{equation}\label{eq:came_layer}
  \mathcal{X}
  \leftarrow
  \mathcal{X} + \operatorname{GCA}(\operatorname{GLN}(\mathcal{X})),
  \qquad
  \mathcal{X}
  \leftarrow
  \mathcal{X} + \operatorname{FFN}_{\mathrm{grade}}(\operatorname{GLN}(\mathcal{X})),
\end{equation}
其中 $\operatorname{FFN}_{\mathrm{grade}}$ 表示逐阶独立的 MLP，
不会混合不同阶的系数。

\subsection{几何层归一化}\label{sec:gln}
% ------------------------------------------------------------

实现采用阶别分离的几何归一化。
对 $r$ 阶分量 $\langle\mathcal{X}\rangle_r$，
先计算
\begin{equation}\label{eq:gln_norm}
  \rho_r
  =
  \sqrt{
    \left|
      \bigl\langle
        \langle\mathcal{X}\rangle_r\,
        \widetilde{\langle\mathcal{X}\rangle_r}
      \bigr\rangle_0
    \right|
    + \epsilon
  }.
\end{equation}
归一化结果为
\begin{equation}\label{eq:gln_output}
  \operatorname{GLN}(\mathcal{X})
  =
  \sum_{r=0}^{4}
  \gamma_r\,
  \frac{\langle\mathcal{X}\rangle_r}{\rho_r},
\end{equation}
其中 $\gamma_r\in\mathbb{R}$ 为每一阶独立的可学习缩放参数。
与常规 LayerNorm 不同，当前实现\emph{不使用非标量偏置项}，
以避免向量、二向量或三向量通道被固定偏移破坏其几何变换性质。

% ------------------------------------------------------------
\subsection{软等变正则化}\label{sec:equiv_reg}
% ------------------------------------------------------------

尽管点云 MPE 的 1--3 阶通道已采用闭式几何构造，
训练中仍保留软等变正则化以约束整个网络。
对每个 mini-batch 采样一个随机 motor
\begin{equation}\label{eq:sample_motor}
  \mathcal{M}_g
  =
  T(\mathbf{t}_g)\,R(\boldsymbol{\omega}_g),
\end{equation}
其中 $\boldsymbol{\omega}_g\sim\mathcal{N}(0,\sigma_\omega^2 I_3)$，
$\mathbf{t}_g\sim\mathcal{N}(0,\sigma_d^2 I_3)$。
实现中默认 $\sigma_\omega=\pi/6$、$\sigma_d=1.0$。

记 $\Phi$ 为堆叠的 CAME 主干输出的 latent multivector。
对于多模态输入，随机 motor 仅作用于点云坐标，
图像和文本 token 保持不变。
等变正则化写为
\begin{equation}\label{eq:equiv_loss}
  \mathcal{L}_{\mathrm{equiv}}
  =
  \mathbb{E}_{\mathcal{M}_g}
  \Bigl[
    \bigl\|
      \Phi(\mathcal{M}_g\!\cdot\!\mathcal{X}^{\mathrm{pc}},
      \mathcal{X}^{\mathrm{img}},
      \mathcal{X}^{\mathrm{txt}})
      -
      \mathcal{M}_g\!\cdot\!
      \Phi(\mathcal{X}^{\mathrm{pc}},
      \mathcal{X}^{\mathrm{img}},
      \mathcal{X}^{\mathrm{txt}})
    \bigr\|_2^2
  \Bigr].
\end{equation}
若仅使用点云输入，上式自然退化为单模态版本。

总训练目标为
\begin{equation}\label{eq:total_loss}
  \mathcal{L}_{\mathrm{total}}
  =
  \mathcal{L}_{\mathrm{task}}
  +
  \lambda\,\mathcal{L}_{\mathrm{equiv}}.
\end{equation}
实现采用线性预热策略：
在前 $T_w$ 个优化 step 内，将 $\lambda$ 从 $0$ 线性增长至目标值 $\lambda_{\max}$，
随后保持不变。
这一策略先让模型学习任务相关表征，再逐步强化空间等变约束。
